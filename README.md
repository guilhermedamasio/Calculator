# Calculadora Simples

Basicamente eu fiz uma calculadora SIMPLES, tem exatamente 5 funções matemáticas ( %, *, /, -, + ).

Todos os botões são funcionais, inclusive tem a opção de ERROR quando tenta dividir por 0.

O botão "C" limpa todo o menu.
O botão "CE" apaga apenas um caracter, é como se fosse um backspace que existem em calculadoras melhores.
E o restante dos botoões são os números, além de obviamente os botões para as funções.
Essa calculadora também calcula números com vírgula, então é possível calcular DOUBLES.
